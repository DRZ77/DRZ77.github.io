前言

你好，朋友！听说你想学C语言，那你很厉害了。作者也想变得更加优秀，所以在学完C语言网课后想写这么一篇博文，既是对自己所学知识的复习巩固，也为想要学习C语言的同学提供一些素材和思路。这篇博文主要以知识点+代码练习的形式为大家呈现，同时欢迎大家的批评与指正，让我们共同进步。另外，想要在B站上学习的同学如果时间充裕的话我推荐黑马程序员的课程，内容详细易懂。链接放这里了。

1.C语言入门

1.1 环境的搭建

1.1.1 VS

全名Microsoft Visual Studio 2022,简称VS,微软发布的一款较为好用的软件，最新版是2026版，当然也有其他版本号，这里推荐最新版（作者下载时没有2026版）,越新的版本越安全好用。如果你学习C语言的目的是为了个人使用和企业开发，这里作者强烈推荐VS，因为VS更加智能，开发效率更高。下面的代码练习也是用VS实现的。
以下是下载教程：

  1. 打开官网https://www.microsoft.com/，下滑到页面最下面，找开发人员与IT这一栏,点visual studi,直接免费下载
  2. 这里下载的是安装包，我们再继续安装，在弹出来的选项中，在工作负荷中勾选使用C++的桌面开发，其余的暂时不要勾选，在单个组件这一栏的组件是默认的，但是以防万一查看windows 11 SDK是否勾选，如果没有将其勾选，语言包勾选中文（简体），安装位置的话可以看到这三项加一起有打十几GB了，建议不要放在C盘，可以自己定义。
  3. 最后点击安装，下载时间较长请耐心等待，下载后软件不在桌面，需要在菜单栏里面寻找，可以将其固定在桌面或任务栏

1.1.2 VC

全名Microsoft Visual C++ 2010(学习版)，简称VC。如果大家学习C语言的目的是为了考试，比如计算机二级，专升本考试，期末考试等，这里作者建议使用VC，因为不仅许多考试的要求就是用VC，就连我们的教材，谭浩强老师编著的《C程序设计》一书中提到的开发工具也是VC。但是这个软件微软已经停止更新了，所以在官网里是找不到的，想要下载的朋友只能在别处找了。

1.2 Hello World!

1.2.1 输入并运行你的第一行代码

众所周知，不管是是python，java；还是C语言，PHP。咱们的第一行代码都是Hello World!,这个规矩来自C语言之父–丹尼斯·麦卡利斯泰尔·里奇（Dennis MacAlistair Ritchie）这位大佬编写的一本著作《C语言程序设计》，在这本书中的第一个程序代码就是展示一段话Hello World!，所以将这行代码作为程序员的启蒙代码，接下来让我们使用VS来做一个代码实现吧：

  1. 首先打开我们心爱的VS，创建新项目，选择控制台应用，下一步，项目名称可以自己定义，项目位置也不做要求，因为我们自己写的代码内存通常只有几KB。
  2. 创建完项目后可以看到原本是有代码的，我们在右侧源文件中找到刚刚创建的项目，右键点击移除，在弹出的提示中点击删除，不然我们接下来写的代码会报错
  3. 右键源文件点击添加–>新建向–>显示所有配置，在名称一栏中填写Hello World!.c(注意，这里我们学习的是C语言而不是C++，所以文件后缀是.c而不是.cpp),最后点击添加。
  4. 接下来将下面的代码输入到刚刚创建的项目里（不要忘记保存代码，可以ctrl+s,也可以直接点击保存按钮），最后点击上册工具栏中的空心三角运行代码。
