2. C语言入门

2.1. 环境的搭建

2.1.1. VS

全名Microsoft Visual Studio 2022,简称VS,微软发布的一款较为好用的软件，最新版是2026版，当然也有其他版本号，这里推荐最新版（作者下载时没有2026版）,越新的版本越安全好用。如果你学习C语言的目的是为了个人使用和企业开发，这里作者强烈推荐VS，因为VS更加智能，开发效率更高。下面的代码练习作者是用2022版的VS实现的。
以下是下载教程：

  1. 打开官网https://www.microsoft.com/，下滑到页面最下面，找开发人员与IT这一栏,点visual studi,直接免费下载
  2. 这里下载的是安装包，我们再继续安装，在弹出来的选项中，在工作负荷中勾选使用C++的桌面开发，其余的暂时不要勾选，在单个组件这一栏的组件是默认的，但是以防万一查看windows 11 SDK是否勾选，如果没有将其勾选，语言包勾选中文（简体），安装位置的话可以看到这三项加一起有打十几GB了，建议不要放在C盘，可以自己定义。
  3. 最后点击安装，下载时间较长请耐心等待，下载后软件不在桌面，需要在菜单栏里面寻找，可以将其固定在桌面或任务栏

2.1.2. VC

全名Microsoft Visual C++ 2010(学习版)，简称VC。如果大家学习C语言的目的是为了考试，比如计算机二级，专升本考试，期末考试等，这里作者建议使用VC，因为不仅许多考试的要求就是用VC，就连我们的教材，谭浩强老师编著的《C程序设计》一书中提到的开发工具也是VC。但是这个软件微软已经停止更新了，所以在官网里是找不到的，想要下载的朋友只能在别处找了。

2.2.2. 代码解析

以下解析理解为主，无需强行背，用多了就熟悉了。

  1. int main上面的代码我们称为预处理，就是将运行代码前要用到的东西做一个声明，#include就是找的意思，找的内容就是这个头文件<stdio.h>，全称为StandardInputandOutput（标准的输入和输出），它就包含了下面的printf(输出语句)，,也就是说如果没有声明这个头文件VS是识别不了printf的,代码就会报错。我们后续也会渐渐了解其他头文件，比如与数学计算相关的<math.h>,与时间相关的<time.h>,目前我们先掌握<stdio.h>就好。
  2. 预处理下面的我们称为业务代码，就是我们要让计算机做的事，第一行int main ()表示程序的主入口，计算机找到这个入口就会从上到下依次运行代码，C语言规定主入口只能叫做main,这个是不呢个修改的，()目前我们就把它当做一个必不可少的格式就行，int表示整个程序运行的结果是个整数，{}表示一个范围，我们有关这个程序的所有代码都要写在这个带括号里面。
  3. printf表示输出语句，就是告诉计算机我们要输出一段内容了，而这个内容就是()中的内容，这一长串文本需要用””(引号)引起来，最后 ；(分号)表示语句结束。
  4. return表示返回，整数0作为返回值与int对应，如果返回0.1,0.2这样的小数代码就会报错，目前我们只用知道返回0表示程序正常结束，返回非0表示程序异常结束，将return 0作为一个格式放在代码最后就好。其实经作者多次测试，这个return 0写不写没多大事，但是作者还是建议养成一个好习惯。

  • 关键字是指被C语言富裕了特定含义的英文单词，比如上满提到的int(整数)，renturn(程序结束)，诸如此类的关键字在C语言中共有32个，随着课程深入我们都会进行学习。

3.5. 进制

3.5.1. 二进制

由0和1组成，根据逢二进一可知，十进制中的2表示二进制中的10,3表示11,4表示100。代码以0b开头（b的大小写不做要求，一版为小写），如int a = 0b10,就是给a赋值为2.

3.5.2. 八进制

对于大数据二进制太长不方便阅读，八进制应运而生。将二进制中的数据每三个分为一组，最小的000就是0，最大的111就是7，所以八进制的取值范围0~7。代码以0开头，如，int a = 010是将8赋值给a

3.5.3. 十六进制

随着时代的发展，八进制被十六进制取代，十六进制将4个二进制分为一组得到，最小的仍是0，最大的1111是15.代码以0x开头，有09与af组成，af分别对应1015。如，int a=0x10是将16赋值给a

3.5.4. 进制转换

  • 任意进制转十进制，公式：系数*技术的权次幂，最后相加。（权：从右至左依次为0,1,2···）
  • 十进制转任意进制，用到除基取余法（不断除以基数知道商为0，在将余数倒着拼接）

3.7. 标识符

  • 标识符：代码中我们自己起的名字，如变量名，函数名的名等
  • 命名的硬性规则：1.由数字，字母下划线（_）组成；2.不能以数字开头；3.不能是关键字（小写且有颜色标记）；4.区分大小写；
  • 命名的软性建议：尽量以英文命名，做到见名如意；变量名全部小写；代码文件名全部小写，单词之间用下滑线隔开，开头可以用数字

14.补充

14.1. 快捷键

14.1.1. 常见快捷键

  • Ctrl + C：复制
  • Ctrl + V：粘贴
  • Ctrl + X：剪切
  • Ctrl + Z：撤销
  • Ctrl + A：全选
  • Ctrl + S：保存
    以上的快捷键对提高我们编写代码的效率很有帮助，建议牢记。

14.1.2. 注释快捷键

  • 加上注释：选中内容–>ctrl+k,ctrl+c 或者，选中内容–>ctrl+shift+/。每次写完代码要创建新项目时需要注释掉当前项目的代码，作者最爱用的小连招是：ctrl+A,ctrl+K,ctrl+C.
  • 取消注释：选中内容–>ctrl+k,ctrl+c
